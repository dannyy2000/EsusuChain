# EsusuChain Frontend

A beautiful, modern web interface for EsusuChain - automated savings circles on Flow blockchain.

## Features

- **Landing Page**: Stunning hero section with animated backgrounds and feature highlights
- **Dashboard**: View all your active circles, stats, and recent activity
- **Create Circle**: Step-by-step circle creation with smart validation
- **Circle Details**: Deep dive into circle progress, members, and automated execution

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **Lucide React** - Beautiful icons

## Getting Started

### Installation

```bash
cd frontend
npm install
```

### Development

```bash
npm run dev
```

Visit http://localhost:5173

### Build for Production

```bash
npm run build
npm run preview
```

## Design System

### Colors

- **Primary (Green)**: `#22c55e` - Success, growth, savings
- **Flow (Teal)**: `#14b8a6` - Flow blockchain branding
- **Background**: Dark mode (`#030712`)
- **Glass Effects**: `backdrop-blur` with alpha transparency

### Typography

- **Font**: Inter (Google Fonts)
- **Headings**: Bold, large sizes
- **Gradient Text**: Flow → Primary color gradient

## Pages

1. **LandingPage** - Marketing page with features
